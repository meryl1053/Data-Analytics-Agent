# AI Data Analysis Agent

## Quick Start

1. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Run the application:**
   ```bash
   streamlit run integrated_app.py
   ```

3. **Upload data and analyze!**
